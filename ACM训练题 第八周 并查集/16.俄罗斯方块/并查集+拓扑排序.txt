题目：Sample Input
3 3 
0 > 1 
1 < 2 
0 > 2 
4 4 
1 = 2 
1 > 3 
2 > 0 
0 > 1 
3 3 
1 > 0 
1 > 2 
2 < 1

Sample Output
OK
CONFLICT
UNCERTAIN
（难）
分析：
①使用并查集 拓扑排序
②并查集是用来处理=号情况的 两个元素相等时入度为0，需要合并
③ 拓扑排序（首先对于拓扑排序要有很深的理解）
  出现信息不完整的表现时同时出现两个或两个以上的元素 入度为0，且根是自己
  （因为如果说=合并了 我们可以这样想 这样就意味这这两个点是一模一样的）
  出现冲突的表现时出现环了（这也是拓扑排序中的冲突出现情况），这样造成拓扑排序的元素少于需要排序的元素，有的元素永远没有进行排序，是一个死循环
